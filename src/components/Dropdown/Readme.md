Default Dropdown

```js
<Dropdown>
  <DropdownActivator>Click here!</DropdownActivator>
  <DropdownMenu>Any dropdown content!</DropdownMenu>
</Dropdown>
```

Dropdown with menu list items

```js
<Dropdown>
  <DropdownActivator>Click here!</DropdownActivator>
  <DropdownMenu displayAsMenuItems>
    {
      ['item', 'item', 'item', 'item']
    }
    <a href="#">link</a>
  </DropdownMenu>
</Dropdown>
```
